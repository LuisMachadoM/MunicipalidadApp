# Frontend - Municipio en Línea

Frontend base en React + Vite para el proyecto Municipio en Línea.

## Instalación
```bash
npm install
cp .env.example .env
npm run dev
```

## Requiere
- Backend corriendo en `http://localhost:3001`
- Base de datos cargada

## Funcionalidades base
- Inicio
- Registro ciudadano
- Login
- Listado de trámites
- Ver formulario por trámite
- Enviar solicitud
- Ver mis trámites
- Panel funcionario
